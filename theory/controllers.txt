Для того, чтобы работать с  ГЕТ или ПОСТ параметрами в контроллере, необходимо в качестве аргумента
в метод передать экземпляр класса Request, и потом работать с ним в методе, используя его методы:

        public function actionAbout(Request $request){
            return view("aboutView",[
                "name"=>$request->get("name","anonim")
            ]);
        }

Для того, чтобы работать с параметрами, переданными через url, мы должны передать их в качестве аргумента
в метод контроллера, обратите внимание, если параметр не обязателен, то необходимо обязательно указать
значение по умолчанию:

    public function actionProfile($name="anonim"){
            return view("profileView",[
               "name"=>$name
            ]);
        }
